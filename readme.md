# mihiraki

E-book Viewer WebComponent.

## Examples

```
<!DOCTYPE html>
<html>
  <head>
    <title>Mihiraki WebComponents example</title>
    <script type="module" src="mihiraki.umd.js"></script>
  </head>
  <body>
    <mihiraki-viewer src="./page%d.png" total-pages="5">
    </mihiraki-viewer>
  </body>
</html>
```

See example directory for more information.
